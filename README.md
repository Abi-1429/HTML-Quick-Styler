# ⚡ HTML Quick-Styler

### 🚀 AI-Powered HTML Page Generator using IBM Granite 3.3 2B

Generate **production-ready HTML & CSS pages in seconds** with AI-assisted layouts, responsive design, and beautiful color palettes.

---

## ✨ Features

* 🧠 **AI-Generated Content** powered by IBM Granite 3.3 2B
* 🎨 **8 Built-in Color Palettes**
* 📱 **Fully Responsive Layouts**
* ⚡ **Instant HTML/CSS Generation**
* 👁️ **Live Page Preview**
* 📦 **Production-Ready Export**
* 🧩 **Smart Section Detection** from description

## 🧠 AI Section Detection

The generator automatically detects sections from your description.

| Keyword                 | Generated Section |
| ----------------------- | ----------------- |
| `hero` / `banner`       | Hero section      |
| `features` / `services` | Features grid     |
| `about` / `team`        | About section     |
| `portfolio` / `gallery` | Portfolio         |
| `testimonials`          | Testimonials      |
| `pricing` / `plans`     | Pricing table     |
| `contact` / `form`      | Contact form      |
| `footer`                | Footer            |

## ⚙️ Installation

Install dependencies:

```bash
pip install gradio transformers torch accelerate sentencepiece
```

---

## ▶️ Run the App

```bash
python app.py
```

The app will launch and generate a **public Gradio link**.

Example:

```
Running on public URL: https://xxxx.gradio.live
```
